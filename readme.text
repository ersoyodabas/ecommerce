aşağıdaki kod veritabanı scaffold kodudur.

dotnet ef dbcontext scaffold "Server=AYLIN;Database=ecommerce;Trusted_Connection=True;TrustServerCertificate=True;" Microsoft.EntityFrameworkCore.SqlServer -o Models -c EcommerceDbContext --force --use-database-names
